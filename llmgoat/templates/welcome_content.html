LLMGoat

This project is an open source, deliberately vulnerable environment created by SECFORCE to learn about LLM-specific risks based on the OWASP Top 10 for LLM Applications.
Each challenge from the left menu simulates a real-world vulnerability so you can easily learn, test and understand risks associated with large language models.
No goats were harmed during the creation of this tool... we can't say the same for our own LLMs, unfortunately.
